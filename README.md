# Olist-Ecommerce-Analysis
Data-driven Power BI dashboard built on Olist e-commerce dataset, analyzing sales performance, customer behavior, delivery efficiency, and revenue trends using advanced DAX measures and relational data modeling.

# Olist E-Commerce Sales Analysis (Power BI)

##  Project Overview
This project analyzes Brazilian e-commerce data using Power BI.

## Key Insights
- Majority customers are one-time buyers (~95%)
- Delivery delays negatively impact customer ratings
- Revenue is concentrated in top product categories
- Sales show consistent growth trend

## KPIs
- Total Revenue
- Total Orders
- Total Customers
- Average Rating
- On-Time Delivery %

## Tools Used
- Power BI
- DAX
- Data Modeling
